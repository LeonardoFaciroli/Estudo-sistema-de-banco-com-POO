Este projeto é um aplicativo de linha de comando que simula um pequeno sistema bancário utilizando PHP e orientação a objetos. Ele permite criar contas correntes e poupanças, realizar depósitos, saques, transferências, aplicar rendimento mensal na poupança e exibir extratos completos com o histórico de transações. O código demonstra o uso de herança, polimorfismo, classes abstratas, exceções personalizadas e organização baseada em regras reais de um banco. A classe abstrata Conta define o comportamento comum, enquanto ContaCorrente e ContaPoupanca implementam suas particularidades, principalmente no cálculo das tarifas. O sistema registra todas as operações em objetos de transação e fornece ao usuário um menu interativo no terminal.
